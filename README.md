# sakura

Keycloak login theme for [miyazakitakara.ovh](https://miyazakitakara.ovh).

![preview](docs/preview.gif)

## Installation

```bash
git clone https://github.com/MiyazakiTakara/sakura.git /opt/keycloak/themes/sakura
```

Activate: **Realm Settings → Themes → Login Theme → sakura**
